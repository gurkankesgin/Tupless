## Getting Started

there is two simple steps.

#### Step 1. Add the JitPack repository to your build file

```sh
allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
 ```


####   Step 2. Add the dependency

```sh
dependencies {
	        implementation 'com.github.gurkankesgin:Tupless:0.1.0'
	}
```
